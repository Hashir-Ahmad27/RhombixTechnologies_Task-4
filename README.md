# Library Management System - Rhombix Technologies

A console-based **Library Management System** developed during my internship at **Rhombix Technologies**. This project demonstrates key **C++ programming** concepts such as object-oriented programming, file handling, class structures, inheritance, constructors, and user interaction.

---

## Features
- Users can search for books, issue and return them, and view due dates.  
- Administrators can add new books, delete existing ones, and manage book records.  
- Stores data using files for persistence across sessions.  
- Provides a simple console interface for easy interaction.  

---

## Technologies Used
- **Programming Language:** C++  
- **Development Environment:** Any C++ compiler (e.g., Code::Blocks, Visual Studio, g++, etc.)
